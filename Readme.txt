Inventory Management System  - Console application 

Named as GildedRose

Project is console application, Build and run by visual studio 2017 or higher. it has unit test project, it uses Nunit Framework.

Run:
Start: Displays inventory list (Task sample)
Enter: Type "DayEnd" and Press Enter key to stimulate end of the day result.
        -> To view next day result Press Enter key.
        -> Keen Press Enter key to view consecutive day's result.
Exit:  Type "Exit" to exit the screen


There is sample inventory list stored under GildedRoseLib project
    GildedRoseLib->Access->Repos.cs class
    Quality and SellIn values can be edited from Repos.cs class.
